Gradient Descent Algorithm

Update Rule
$$x_{n+1} = x_n - \eta \nabla f(x_n)$$
  • Learning Rate: Step size parameter η controls convergence speed
  • Gradient: Points in direction of steepest ascent
  • Convergence: Iterative approach to local minimum

Sigmoid Activation

Sigmoid Function
$$\sigma(x) = \frac{1}{1 + e^{-x}}$$
  • Range: Output bounded in (0, 1)
  • Derivative: σ'(x) = σ(x)(1-σ(x))
  • Use: Binary classification, neural network gates

Algorithm Flow

Initialize x₀ → Compute ∇f(x) → Update x → Check Convergence → Return x*

Key Insight

Gradient descent forms the backbone of neural network training. The algorithm enables optimization of millions of parameters through backpropagation, making modern deep learning possible.

Unit Ball Projection

Feasible Region
$$D = \{(x,y) \in \mathbb{R}^2 : x^2 + y^2 \leq 1\}$$
Projection
$$\text{Proj}_D(P) = \begin{cases} P & \|P\| \leq 1 \\ \frac{P}{\|P\|} & \|P\| > 1 \end{cases}$$

Box Projection

Feasible Region
$$S = \{(x,y) \in \mathbb{R}^2 : -1 \leq x, y \leq 1\}$$
Projection
$$\text{Proj}_S(P) = \text{clip}(P, -1, 1)$$

PGD Algorithm

Start P₀ → Compute ∇f → P - η∇f → Project → Repeat

Constrained Optimization

Projected gradient descent ensures feasibility at each iteration by projecting back onto the constraint set, enabling optimization with hard bounds on variables.

Finite Differences

Central Difference
$$f'(x) \approx \frac{f(x+h) - f(x-h)}{2h}$$
  • Forward: (f(x+h) - f(x)) / h
  • Backward: (f(x) - f(x-h)) / h
  • Central: Second-order accurate

Interpolation

Lagrange Polynomial
$$L(x) = \sum_{i} y_i \prod_{j \neq i} \frac{x - x_j}{x_i - x_j}$$
  • Splines: Piecewise polynomials
  • Newton: Divided differences
  • Continuity: Smoothness conditions

Simpson's Rule

Composite Simpson
$$\int_a^b f(x)dx \approx \frac{h}{3}\left[f_0 + 4\sum_{odd} + 2\sum_{even} + f_n\right]$$
  • Parabolic: Quadratic approximation
  • Panels: Even number required
  • Error: Proportional to h⁴

Gaussian Quadrature

Weighted Sum
$$\int_{-1}^{1} f(x)dx \approx \sum_{i=1}^{n} w_i f(x_i)$$
  • Nodes: Legendre polynomial roots
  • Weights: Optimal coefficients
  • Exactness: Polynomials up to 2n-1

Monte Carlo Integration

For high-dimensional integrals, Monte Carlo methods provide efficient estimation by random sampling, with error decreasing as 1/√N regardless of dimension.
