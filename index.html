Gradient Descent Algorithm

Update Rule $$x_{n+1} = x_n - \eta \nabla f(x_n)$$
  • Learning Rate: Step size parameter \(\eta\)
  • Gradient: Direction of steepest ascent
  • Convergence: Iterative minimization

Sigmoid Activation

Sigmoid Function $$\sigma(x) = \frac{1}{1 + e^{-x}}$$
  • Range: Output bounded in (0, 1)
  • Derivative: \(\sigma'(x) = \sigma(x)(1-\sigma(x))\)
  • Use: Binary classification, gates

Algorithm Flow

Initialize x0
→
Compute ∇f(x)
→
Update x
→
Check Convergence
→
Return x*

Unit Ball Projection

Feasible Region $$D = \{(x,y) \in \mathbb{R}^2 : x^2 + y^2 \leq 1\}$$
Projection $$\text{Proj}_D(P) = \begin{cases} P & \|P\| \leq 1 \\ \frac{P}{\|P\|} & \|P\| > 1 \end{cases}$$

Box Projection

Feasible Region $$S = \{(x,y) \in \mathbb{R}^2 : -1 \leq x, y \leq 1\}$$
Projection $$\text{Proj}_S(P) = \text{clip}(P, -1, 1)$$

Finite Differences

Central Difference $$f'(x) \approx \frac{f(x+h) - f(x-h)}{2h}$$
  • Forward: \(\frac{f(x+h) - f(x)}{h}\)

  • Central: Second-order accurate

Interpolation

Simpson's Rule

Composite Simpson $$\int_a^b f(x)dx \approx \frac{h}{3}[f(a) + 4\sum_{odd} + 2\sum_{even} + f(b)]$$
  • Parabolic: Quadratic approximation
  • Even Panels: Required for composite
  • Error: Proportional to h⁴

Gaussian Quadrature

Weighted Sum $$\int_{-1}^{1} f(x)dx \approx \sum_{i=1}^{n} w_i f(x_i)$$
  • Nodes: Legendre polynomial roots
  • Weights: Optimal coefficients
  • Exactness: Polynomials up to 2n-1

Monte Carlo Integration

For high-dimensional integrals, Monte Carlo methods provide efficient estimation by random sampling, with error decreasing as 1/√N regardless of dimension.
